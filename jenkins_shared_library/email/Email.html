Package: jenkins_shared_library.email

[Groovy] Class Email

    • jenkins_shared_library.email.Email

    • Class to send email notifications.

      Example:
           def m = new Email(this)
     m.send(
         subjectTag   : 'FAILURE',
         to:          : 'also-to-me@gmail.com',
         body         : '<p>Test failed!</p>',
         addProviders : true
     )
